# VartoGhostRemover
VartoGhostRemover — це інструмент безпеки, заснований на технології активного обману (Active Deception) для виявлення та нейтралізації шкідливого ПЗ. Проєкт фокусується на виявленні стілерів та прихованих процесів, що намагаються отримати доступ до конфіденційних даних.

## Концепція
Замість пасивного сканування за сигнатурами, VartoGhostRemover розгортає систему "пасток" (honey-files). При спробі шкідливого процесу взаємодіяти з такими файлами, система аналізує аномальну поведінку (навантаження CPU/IO) та автоматично нейтралізує загрозу.

## Основні можливості
Deception Engine: Створення та відстеження фальшивих цілей для виявлення стілерів.

Behavioral Analysis: Моніторинг системних ресурсів у реальному часі для виявлення підозрілої активності.

Surgical Removal: Повне видалення слідів ПЗ, включаючи ключі реєстру, автозавантаження та фонові служби.

Performance Focused: Написано на Rust для мінімального споживання системних ресурсів.

## Технічний стек
Backend: Rust (низькорівнева робота з системними API).

Frontend: Tauri (React/Svelte) для забезпечення легкого та чутливого інтерфейсу.

OS Support: Windows (агент моніторингу), Linux (основна аналітична платформа).
